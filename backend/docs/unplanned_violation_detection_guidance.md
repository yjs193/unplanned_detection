# 无计划作业违规检测指导手册（合景输变电工程原型版）

版本：v0.1  
适用模块：作业票智能解析 -> 现场媒体调取 -> 视觉理解 -> 违规检测智能体  
适用场景：220千伏合景输变电工程在线试点，以及同类变电站新建、扩建、改造、线路接入和电缆工程场景。

## 1. 手册用途

本手册用于约束“违规检测智能体”的判断边界。智能体不得仅凭作业票字段本身判定“无计划作业”，必须把作业票解析结果与现场图片、视频或视觉理解文本进行一致性比对。

核心判定对象是：现场实际发生的时间、地点、人员、机械、作业动作、设备对象、施工状态和安全控制措施，是否与作业票允许的计划侧事实一致。

## 4. 总体检测逻辑

违规检测智能体按以下顺序执行，不得跳步：

1. 状态门控：仅对 `plan_status=开工中` 的作业票执行现场一致性检查。待开工票若现场出现施工，判为高风险；已完工票若现场仍出现同类或其他施工，判为高风险。
2. 时间比对：现场图片/视频时间必须落在作业票计划时间窗口内。窗口外施工属于疑似无计划作业。
3. 地点比对：摄像头、无人机航线、画面场景线索必须匹配作业票地点、房间、间隔、塔位、沟道或站区部位。
4. 作业内容比对：视觉识别出的动作、设备对象和施工工序必须属于票面作业内容或其合理子工序。
5. 人员与机械比对：现场人数、机械、车辆、吊装设备、动火设备、开挖设备等不得明显超出票面允许范围或作业类型需要。
6. 安全前提比对：若票面涉及停电、运行区域、邻电、动火、吊装、高处、深基坑等，视觉结果中应出现相应隔离、防护、警戒、监护或许可证据。
7. 异常聚合：同一时间段连续多帧出现同类偏差时，合并为一个事件；不要按每一帧重复报同一问题。

## 5. 通用无计划作业判定类别

| 编号 | 类别 | 判定条件 | 风险等级 |
|---|---|---|---|
| G01 | 无票施工 | 没有匹配作业票，但现场出现施工人员、机械、动火、开挖、吊装、拆除等明显作业 | 高 |
| G02 | 状态不符 | 作业票待开工/已完工，但现场出现施工 | 高 |
| G03 | 时间不符 | 现场施工时间早于计划开始或晚于计划结束 | 高 |
| G04 | 地点不符 | 画面地点、摄像头点位或视觉场景与票面地点不一致 | 高 |
| G05 | 作业动作不符 | 票面为A作业，现场识别为B作业，且B不是A的合理子工序 | 高 |
| G06 | 设备对象不符 | 票面对象为主变基础/GIS室/电缆沟等，现场对象为其他设备或无关房建区域 | 高 |
| G07 | 未授权特殊作业 | 现场出现动火、吊装、开挖、拆除、带电邻近、高处、深基坑等，票面未包含 | 高 |
| G08 | 人员规模异常 | 现场人数明显高于票面或作业类型合理人数，且无法解释为同票相关班组 | 中 |
| G09 | 机械设备异常 | 现场出现挖掘机、吊车、泵车、塔吊等大型机械，票面无相应作业 | 高 |
| G10 | 安全措施缺失 | 作业内容匹配，但风险控制措施对应的可见证据明显缺失 | 中/高 |
| G11 | 同区多作业冲突 | 同一监控区域同时出现两个互不相关作业，只有一张票覆盖其中一个 | 高 |
| G12 | 作业阶段错位 | 票面为准备/收尾/材料进退场，现场却为主体施工或拆除开挖 | 中/高 |

## 6. 合景输变电工程作业类型检测规则库

### 6.1 场地平整、施工道路、材料站和临设布置

允许视觉证据：场地清表、围挡、临时道路、材料堆放、施工标识、车辆进出、临设搭建。  
常见无计划情况：

- 票面未包含临设/场地作业，但现场出现推土、压实、临时道路开辟。
- 票面地点为主变基础或GIS室，画面为站外道路、材料站或生活区施工。
- 现场材料大规模进场，但作业票仅允许巡检、测量或收尾。

### 6.2 土方开挖、基坑支护、沟槽开挖与回填

允许视觉证据：挖掘机、渣土车、沟槽/基坑、支护、围蔽、临边护栏、警示灯、土方堆放、回填压实。  
常见无计划情况：

- 票面为电缆敷设或设备安装，现场出现土方开挖、破路、基坑支护。
- 票面为回填恢复，现场仍在继续开挖或扩大基坑。
- 票面作业地点为电缆沟A段，画面开挖点位在其他沟道、道路或站外区域。
- 视觉识别到深基坑、临边、高差，但票面未包含基坑/沟槽风险控制。
- 非计划时间段出现挖掘机作业或渣土转运。

### 6.3 地基与基础、主变基础、GIS基础、电缆沟基础

允许视觉证据：测量放线、破桩头、承台/地梁、钢筋绑扎、模板安装、混凝土浇筑、预埋件、预埋槽钢、基础埋件、回填、养护。  
常见无计划情况：

- 票面为主变基础，现场识别为室内装修、墙体砌筑、设备安装或其他无关作业。
- 票面为钢筋绑扎/支模，现场识别为混凝土浇筑；若票面未覆盖浇筑阶段，判作业阶段不符。
- 票面为GIS室基础埋件，现场识别到主变室、道路、围墙或其他区域施工。
- 视觉识别到吊车、泵车、焊机等，而票面未包含吊装、泵送、动火或对应风险控制。
- 现场出现新增基础、扩基、拆墙、开孔等票面未列动作。

### 6.4 钢筋、模板、高支模与混凝土工程

允许视觉证据：钢筋材料、绑扎、模板、支撑架体、混凝土泵管、振捣、浇筑面、养护。  
常见无计划情况：

- 票面为模板安装，现场进行模板拆除或混凝土浇筑。
- 票面为普通结构施工，现场识别到高支模、大面积支撑体系或高处作业，票面无高支模风险控制。
- 票面为混凝土施工，现场出现钢筋加工、切割、焊接等未覆盖工序。
- 混凝土浇筑画面出现在计划时间外或非票面区域。

### 6.5 脚手架、内外架、高处作业平台

允许视觉证据：脚手架杆件、扣件、脚手板、安全网、连墙件、搭设/拆除人员、高处作业人员。  
常见无计划情况：

- 票面未包含脚手架搭设/拆除，现场出现脚手架搭拆。
- 票面为砌体/抹灰，现场出现外架拆除或高支模拆除，且票面未覆盖该高风险动作。
- 现场高处人员无安全带、安全帽、安全网等可见证据，作为安全违规项。
- 同一区域同时出现脚手架拆除和其他下方交叉作业，票面未说明交叉作业控制。

### 6.6 砌体、抹灰、扇灰、瓷砖铺贴、墙体/隔墙改造

允许视觉证据：砖块、砂浆、砌筑、抹灰、墙面处理、隔墙拆除/新建、瓷砖铺贴、室内装修工具。  
常见无计划情况：

- 票面为电气安装或主变基础，现场识别为室内装修、抹灰、瓷砖铺贴。
- 票面为砌体工程，现场出现电缆敷设、设备安装、动火、开孔等非砌体作业。
- 票面仅允许新建隔墙，现场识别为拆除承重/结构构件或管线迁移。
- 施工地点与票面房间不一致，如票面为接地变室，画面为休息室、工具间或其他楼层。

### 6.7 路面破除、电缆沟、井、槽盒、盖板、预制件埋设与恢复

允许视觉证据：路面切割/破除、沟槽、井室、槽盒、盖板、预制构件、围蔽、回填恢复。  
常见无计划情况：

- 票面为电缆沟土建，现场进行道路大面积破除或管廊拆除且未在票面出现。
- 票面为回填恢复，现场仍在开凿、破除、扩大开挖。
- 现场识别到电缆沟孔洞、穿墙孔新增，但票面未包含开孔/开凿。
- 现场在交通通道或站区主道路作业，但票面地点为室内电缆层或GIS室。

### 6.8 接地网、接地扁钢、焊接、防腐

允许视觉证据：接地扁钢、接地线、焊接、沟槽、接地网敷设、防腐涂刷、焊机、灭火器。  
常见无计划情况：

- 票面未包含接地作业，现场出现接地扁钢敷设、焊接、防腐。
- 票面包含接地，但现场识别到动火焊接，票面 `special_operations` 未包含动火作业。
- 焊接作业未见灭火器、清理可燃物、隔离围挡等视觉证据。
- 接地沟槽位置与票面设备区域不一致。

### 6.9 动火、切割、焊接

允许视觉证据：焊机、气瓶、切割机、火花/烟尘、动火点、灭火器、监护人、隔离带。  
常见无计划情况：

- 票面未包含动火作业，现场出现焊接、切割、火花或气瓶。
- 动火时间、地点与票面不一致。
- 现场动火点周边存在电缆、油池、主变附件、可燃物，但未见隔离和灭火器。
- 动火与其他施工交叉进行，票面未包含交叉作业管控。

### 6.10 吊装、塔吊、吊车、主变就位和重物转运

允许视觉证据：塔吊、汽车吊、吊索具、起吊物、吊装警戒区、指挥人员、设备就位、运输车辆。  
常见无计划情况：

- 票面未包含吊装/起重，现场出现吊车、塔吊或起吊状态。
- 票面为材料进退场，现场出现主变、开关柜、GIS设备等重大设备吊装就位。
- 起吊对象与票面设备对象不一致，如票面为主变附件安装，现场吊装大型构架或其他设备。
- 现场无警戒区、指挥人员或吊装半径控制可见证据。

### 6.11 主变、油浸电抗器、端子箱、风冷箱、中性点设备安装

允许视觉证据：主变本体、附件、套管、油池、端子箱、风冷箱、中性点支架、消防管道、设备基础、吊装/转运。  
常见无计划情况：

- 票面为主变基础土建，现场出现主变本体就位或电气安装。
- 票面为主变附件安装，现场识别为基础开挖、墙体拆除、管线迁移。
- 票面为#3主变室，现场设备对象为其他主变、GIS室或电容器室。
- 现场出现油池拆除/新建、消防管道迁移，但票面未包含相关内容。

### 6.12 GIS室、高压室、电容器室、接地变室、配电装置安装

允许视觉证据：GIS设备、开关柜、电容器组、站用变、接地变、刀闸、互感器、硬母线、软导线、屏柜、槽盒、封堵。  
常见无计划情况：

- 票面为土建基础，现场出现高压柜、GIS、母线、刀闸、互感器安装。
- 票面为设备安装，现场出现土建拆除、开孔、管线迁移或装修作业。
- 票面为10kV高压室，现场识别到110kVGIS室或主变室。
- 现场出现带电邻近设备、运行区域作业，但票面未标记运行区域或邻电作业。
- 现场出现一次/二次电缆接线、屏柜安装，但票面仅覆盖土建或材料转运。

### 6.13 电缆支架、桥架、竖井、槽盒、电缆敷设和接线

允许视觉证据：电缆支架、桥架、槽盒、电缆盘、电缆敷设、二次接线、竖井作业、屏柜、接地线。  
常见无计划情况：

- 票面为电缆支架/桥架安装，现场出现设备吊装、土建开挖、动火或拆墙。
- 票面为二次电缆敷设，现场识别为一次电缆沟开挖或大型电缆牵引。
- 票面为主控室槽盒，现场位置为电缆层、GIS室或高压室。
- 现场进行防火封堵、开孔、切割，但票面未包含封堵/开孔/动火。

### 6.14 防火封堵、消防、通风、排风、防火阀、排油系统

允许视觉证据：防火泥/防火包、封堵孔洞、消防管道、防火阀、排风机、排油检查井、排油管、油池。  
常见无计划情况：

- 票面为电缆施工，现场进行消防管道迁移、排风机安装或排油系统施工。
- 票面为防火封堵，现场出现开凿、拆墙、管线迁移等前置土建作业。
- 票面为消防/通风施工，现场位置或设备对象不在票面列明区域。
- 涉及主变油池、排油管施工但票面未覆盖动火、开挖或临边风险。

### 6.15 照明、动力、水电、通信、视频监控、自动化和二次系统

允许视觉证据：电缆、线管、桥架、屏柜、通信设备、视频监控设备、接线、线缆标识、调试仪器。  
常见无计划情况：

- 票面为照明/动力改造，现场出现一次设备安装或土建开挖。
- 票面为视频监控/通信设备安装，现场出现电缆沟、支架桥架或大型设备转运。
- 票面为二次接线，现场识别为带电一次设备区域作业，且票面未包含邻电/运行区域。
- 现场出现调试仪器和试验接线，但票面未覆盖电气试验或调试。

### 6.16 电气试验、继电保护、自动化调试、通信调试

允许视觉证据：试验仪器、测试线、继保屏柜、调试人员、隔离警示、记录设备。  
常见无计划情况：

- 票面为安装施工，现场进行加压试验、保护调试或自动化联调。
- 票面为调试，现场出现拆装、动火、开孔、土建施工。
- 调试地点或屏柜间隔与票面不一致。
- 运行区域调试未见隔离、警示、监护等证据。

### 6.17 输电线路、塔基、杆塔组立、架线、附件安装

合景试点若包含线路接入或电缆线路部分，也应覆盖以下类型。  
允许视觉证据：塔基开挖、基础浇筑、杆塔组立、抱杆/吊装、导地线展放、紧线、压接、附件安装、跨越架、牵张场。  
常见无计划情况：

- 票面为变电站内作业，现场识别为塔基、杆塔、架线或牵张场作业。
- 票面为塔基基础，现场进行杆塔组立或架线。
- 票面为架线，现场出现基础开挖或道路修筑。
- 现场出现跨越架、封网、牵张设备，但票面未覆盖跨越/架线作业。
- 塔位、线路段、牵张场位置与票面不一致。

## 8. 告警分级建议

| 等级 | 触发条件 | 系统动作 |
|---|---|---|
| 高 | 无票施工、状态不符、时间不符、地点不符、作业内容完全不符、未授权动火/吊装/开挖/拆除/邻电作业 | 生成高风险告警，绑定作业票与媒体证据，进入异常清单 |
| 中 | 作业内容部分匹配但阶段错位、设备对象不完全一致、人员/机械异常、安全措施缺失 | 生成待确认告警，继续调取相邻时间段和相邻机位 |
| 低 | 票面与现场基本一致，仅存在视觉置信度不足或个别安全措施不可见 | 记录为观察项，不阻断闭环 |

## 9. 不应判定为无计划作业的情况

以下情况不得直接判定为无计划作业：

1. 仅凭作业票缺字段、风险等级高、未纳入视频管控，不得判定无计划。
2. 现场未见人员但能看到施工材料或设备，不得直接判定无计划，应输出“画面证据不足”。
3. 视觉模型低置信度识别到不相关对象时，不得直接判高风险，应要求相邻帧或其他机位验证。
4. 票面作业内容包含“相关管线迁移、设备物资进退场、工器具材料进退场”等宽泛内容时，应允许合理的材料搬运、车辆停靠、短时人员进出。
5. 同一作业票包含多个地点或多个房间时，不应因为单帧只看到其中一个地点而判定地点不符。

## 10. 后续系统落地建议

1. 将本手册拆成机器可检索的规则库：`work_type -> allowed_visual_evidence -> violation_patterns -> severity`。
2. 作业票解析智能体应输出 `work_type_codes`、`allowed_actions`、`allowed_objects`、`allowed_locations`、`special_operation_requirements`。
3. 视觉理解模型应输出结构化 JSON，不只输出自然语言摘要。
4. 违规检测智能体先做规则比对，再用大模型生成报告；规则命中结果必须保留，不得被大模型覆盖。
5. 告警应绑定：作业票ID、媒体ID、摄像头、时间戳、异常维度、证据句、截图路径。
